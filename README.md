<h1> Convolution Module </h1>
Module takes 8bit inputs in each clock cycle and stores into internal memory. After this state, module takes convolution then outputs convoluted image, again, in 8 bits. <br> <br>
The filter is:

$$
\begin{bmatrix}
    0 & 1 \\
    1 & 0 \\
\end{bmatrix}
$$
